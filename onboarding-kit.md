### Chapter 2: The 5 Agent Onboarding Workflows

Ready-to-deploy agent configurations for the 5 most common onboarding tasks.

#### Workflow 1: Welcome Sequence Agent
- Triggers automatically when a deal closes (Stripe, HubSpot, or manual webhook)
- Sends personalized welcome email with intake form link
- Follows up if intake form isn't completed in 24 hours
- Sends "getting started" guide tailored to service type
- **Time saved:** 45 minutes per client

#### Workflow 2: Intake & Document Collection Agent
- Sends smart intake form (dynamic — only asks relevant questions based on service type)
- Tracks which documents have been received and which are outstanding
- Sends polite reminders every 48 hours for missing items
- Organizes received documents into a shared folder structure
- **Time saved:** 90 minutes per client

#### Workflow 3: Workspace Setup Agent
- Creates project workspace (Notion/ClickUp/Asana) from templates
- Sets up shared folders with correct permissions
- Populates project timeline with milestones
- Creates client portal access credentials
- **Time saved:** 30 minutes per client

#### Workflow 4: Kickoff Scheduling Agent
- Analyzes both calendars for optimal meeting times
- Sends calendar invitation with agenda template
- Creates pre-meeting briefing document from intake data
- Sends 24-hour reminder with preparation checklist
- **Time saved:** 20 minutes per client

#### Workflow 5: Status & Communication Agent
- Sends weekly status digests to new clients
- Flags overdue items to the project manager
- Generates 30/60/90-day onboarding progress reports
- Escalates stuck items that need human attention
- **Time saved:** 60 minutes per client, ongoing

**Total time saved per client: 4+ hours**
